# Python Solutions

## Running the tests

Make sure you have [pytest](https://docs.pytest.org/) installed. Then from the
root of the `python` folder, run:

```bash
make test # or
pytest    # if you have it installed globally
```

This will discover and run all `test.py` files under `src/`.
