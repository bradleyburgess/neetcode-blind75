# Java Solutions

## Running the tests

Make sure you have [JUnit](https://junit.org/junit5/) and/or
[Maven](https://maven.apache.org/) installed. Then from the root of the `java`
folder, run:

```bash
make test                     # or
mvn -B test --file pom.xml    # with maven directly
```

This will discover and run all `test.py` files under `src/`.
